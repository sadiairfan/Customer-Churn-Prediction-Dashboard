# 📊 Customer Churn Prediction Dashboard

## 📝 Project Overview
This project focuses on building an end-to-end Machine Learning solution to solve the critical business problem of customer retention. Using customer behavioral and demographic data, the predictive model evaluates whether a customer is at high or low risk of churning (leaving the service). To make the insights actionable for business teams, an interactive web application was developed to provide instant predictions based on user inputs.

## 🚀 Key Features
- **Exploratory Data Analysis (EDA):** Deep analysis of customer behaviors, handling missing values, and engineering key features influencing churn.
- **Predictive Modeling:** A trained machine learning model optimized to evaluate customer risk factors accurately.
- **Interactive Streamlit UI:** A clean, easy-to-use interface built using **Streamlit** that allows users to adjust customer parameters via intuitive sliders and inputs.
- **Real-Time Inference:** Instant feedback displaying dynamic success/danger alerts based on predicted risk levels.

## 🛠️ Tech Stack & Tools Used
- **Programming Language:** Python
- **Machine Learning Frameworks:** Scikit-Learn, Pandas, NumPy
- **Model Deployment & Interface:** Streamlit Engine
- **Data Scaling:** Standard Preprocessing Scalers
- **Development Environment:** Google Colab
